📌 Project Overview

This project demonstrates automatic text generation using a pretrained Transformer-based language model (DistilGPT-2) from Hugging Face.

The system takes a user-defined prompt and generates coherent, human-like text, making it useful for applications like content creation, AI assistants, and research tools.

🚀 Key Features
🔹 Pretrained DistilGPT-2 model (no training required)
🔹 Generates multiple text outputs
🔹 Controlled randomness using seed
🔹 Lightweight and fast inference
🔹 Easy-to-use pipeline API
🧠 Model Details
Model: DistilGPT-2
Architecture: Transformer (Decoder-based)
Library: Hugging Face Transformers
Framework: PyTorch

DistilGPT-2 is a compressed version of GPT-2, offering:

Faster performance ⚡
Lower memory usage 💾
Comparable text quality
